Pesquisa Avançada 

O formulário de pesquisa avançada é definido na seção de Definição de Bases de Dados e contém os campos que estão indexados na FST e que são pesquisáveis a partir do índice de pesquisa.

Para fazer uma pesquisa avançada devem ser executados os seguintes passos:

  1. Selecionar o campo objeto da pesquisa
  2. Clicar em Índice para consultar o Índice relacionado ao campo
  3. Selecionar no índice os termos para a pesquisa. Para isso há duas opções:
    • Selecionar o termo e clicar em Pesquisar para fazer a pesquisa diretamente sobre o termo selecionado
    • Selecionar o termo e clicar sobre Transferir o termo selecionado para transferir o termo para o formulário de pesquisa e construir uma expressão de pesquisa que combine informação proveniente de mais de um campo
  4. Se não for utilizado o índice de pesquisa, pode-se entrar na caixa correspondente palavras relevantes da informação a ser localizada. Para que esta opção funcione, é necessário indexar o campo com a técnica 4 (por palavras) na FST. Se não selecionar nenhum campo da lista de seleção de campos pesquisáveis, a pesquisa incidirá sobre todos os campos da base de dados que tenham sido indexados.
  5. Logo que a seleção esteja completada, clicar sobre para fazer a pesquisa na base de dados e localizar os registros, ou sobre para limpar os termos de pesquisa.

Os registros localizados são apresentados na janela da base de dados para facilitar a edição.

As expressões de pesquisa salvas são armazenadas em um arquivo TXT na pasta pfts/[lang] com o nome de search_expr.tab. Este arquivo pode ser administrado través das opções contidas em Atualizar definições de bases de dados do menu principal de catalogação.

A partir da opção para gerar saídas também se pode salvar e recuperar as expressões de pesquisa previamente salvas.

Se for adicionada a uma caixa de de pesquisa masi de uma expressão previamente salva, as mismas são concatenadas com o operador booleano AND
