Carregar um arquivo ISO

Os arquivos ISO que são gerados na exportação são gravados na pasta ./bases/wrk. Para importar um arquivo ISO clique na opção Selecionar que acompanha o ISO a importar.

No caso de desejar substituir a base de dados com o resultado dos registros importados, marque a caixa identificada como Apagar todos os registros antes de importar. Se esta caixa não for marcada, os registros importados serão carregados no final da base de dados.

Se deseja atualizar o arquivo invertido enquanto importa os registros, marque a caixa identificada como Gerar arquivo invertido. Se esta caixa não for marcada, deverá gerar o arquivo invertido após o processo de importação com a opção do menu de Manutenção da base de dados.

Nota importante.: Se a base de dados tem muitos registros, recomenda-se usar o utilitário mx para gerar o arquivo invertido

Se o arquivo ISO que se deseja importar não está no servidor, deve-se primeiro carregar a arquivo antes de iniciar a importação. Para isso utilize a seção Carregar arquivo ISO que é exibida na tela de importação. Uma vez transferido o arquivo, pode importá-lo.

Nota importante.: o arquivo ISO a importar deve ter as mesmas etiquetas e subcampos da base de dados já que não existe uma FST de importação/reformatação para realizar mudanças na estrutura dos campos.

Se o arquivo ISO a carregar for muito grande, deve-se utilizar FTP ou outro meio para carregá-lo ao servidor.

Use a opção Apagar que acompanha cada arquivo ISO apresentado, para apagar do servidor os arquivos ISO já utilizados.
