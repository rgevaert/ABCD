Carregar FDT

Se a etapa anterior terminou com sucesso, aparece a FDT capturada a partir de uma base WinIsis ou CDS/ISIS e convertida para o formato do ABCD. Aparece também a caixa  correspondente para localizar e carregar a FST.

A base de dados não foi criada ainda, e se clicar em Voltar voltará à janela de criação de nova base de dados.

Mais à frente poderá completar a FDT recém criada a fim de aproveitar as opções adicionais que o ABCD oferece para o tratamento da informação
