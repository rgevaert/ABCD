Copiar registros a partir de servidores remotos utilizando o protocolo Z39.50.

Esta opção do ABCD permite catalogar de forma direta registros provenientes de outras fontes disponíveis através do protocolo Z39.50.

No menu  Conectar a:  é apresentada a lista de servidores que estão disponíveis para a localização de bases de dados. Esta lista é mantida através da opção Configurar cliente Z39.50 disponível (para o administrador) no menu principal do ABCD.

O menu Tabela de conversão contém os esquemas que foram definidos para converter os registros a partir do formato MARC para a definição de campos da base local que vai receber a informação. Esta lista é mantida através da opção Configurar cliente Z39.50 disponível (para o administrador) no menu principal do ABCD. Se não for selecionado nenhum esquema de conversão, é utilizada a FDT da base de dados local para determinar os campos que serão copiados a partir da base de dados remota.

Utilize as opções de pesquisa para estabelecer os critérios a serem aplicados para localizar a informação que deseja catalogar.

Na caixa Mostrar indique quantos registros devem ser usados para configurar as páginas com os resultados obtidos

Tentar a conexão xx vezes estabelece quantas vezes o sistema tentará estabelecer uma conexão no caso de o servidor não puder ser localizado.

Em condições normais, os registros transferidos criam novos registros na base de dados local que recebe a informação.  Para transferir a informação a um registro já existente siga o seguinte procedimento:

  1. Localize na base local o registro já existente que pretende preencher/completar com dados da catalogação Z39.50.
  2. Depois de ter encontrado o registro, clique no botão editar na barra de ferramentas. 
  3. Quando é exibido o formato para a edição do registro, clique no botão que chama o cliente Z39.50 que fica na barra de ferramentas do registro. Se fez alterações no registro, deve salvá-lo antes de chamar o cliente catalogação Z39.50 já que de outra forma as alterações realizadas serão perdidas
  4. Selecione a base de dados a partir da qual irá catalogar, o formato de conversão (se necessário) e faça a busca
  5. Localize o registro que deseja capturar nos resultados apresentados.
  6. Clique no ícone de captura que fica no canto esquerdo superior do registro. As informações obtidas a partir do servidor Z39.50 são copiadas para o registro selecionado:
    • Convertendo os dados aplicando o formato de conversão (se houver)
    • Transferindo os dados para os campos da base de dados local, sempre e quando estes estiverem vazios, ou seja, esta ação de cópia não substitui as informações existentes no registro
  7. Salve o registro da base de dados local
