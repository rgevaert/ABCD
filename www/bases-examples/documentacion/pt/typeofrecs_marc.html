Tipos de Registro (Formato Marc)

Esta opção ajuda a definir os tipos de registro que serão manipulados pela base de dados quando esta utiliza o formato MARC.

O tipo de registro é obtido do campo 3006 do Leader e ajuda a identificar o formato do campo fixo (008) a ser utilizado na descrição bibliográfica da obra.

Para definir os tipos de registro são necessários 2 componentes:

  1. Identificar o campo (tag) da base de dados do qual será obtido (ou a armazenar no caso de um novo registro) o tipo de registro correspondente à obra. A versão atual do ABCD permite definir até dois campos de cujo valor dependerá o  formato do campo fixo (008) a exibir na planilha de entrada
  2. Descrever os tipos de registro levando em conta os seguintes parâmetros:
  • Nome da tabela de definição campo (FDT) que contém os elementos a serem utilizados para construir o formulário de entrada que servirá de entrada ao campo fixo relacionado com o tipo de registro
  • Valor que tem os campos identificados em (1) para que sejam considerados pertencentes a esse tipo de registro
  • Descrição do tipo de registro, o qual será exibido ao operador no momento da entrada de dados para que proceda a seleção do tipo correspondente

A fim de manter a consistência na parametrização do ABCD, os campos fixos são definidos através de uma FDT que contém a descrição dos elementos de dados associados ao tipo de registro e com todas as características definidas no procedimento de criação da FDT da base de dados. Para editar a FDT associada ao campo fixo de um tipo de registro, clique no link editar. Como resultado, será aberta uma nova janela com as características dos elementos de dados que compõem o campo fixo.
