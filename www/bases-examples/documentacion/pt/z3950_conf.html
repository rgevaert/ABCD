Configuração das opções para implementar o protocolo Z3950 nos processos de captura de registros bibliográficos


Apresenta três opções:

Servers
Define a lista de servidores que irá aparecer no menu de seleção do formulário de pesquisa. Os seguintes campos devem ser fornecidos

  • Nome do servidor, tal como apresentado no menu de seleção de servidores Z39.50
  • URL a ser utilizada para sua localização
  • Porta habilitada para acesso ao servidor através do protocolo Z39.50
  • Nome da base de dados a ser acessada no servidor remoto
A base de dados de servidores é mantida na pasta bases sob o nome servers

Formato de conversão

As informações dos servidores Z39.50 sempre estão no formato MARC. Esta opção permite definir um formato de conversão para fazer a equivalência entre os campos MARC da base de dados Z39.50 e os campos correspondentes da base de dados local.

A tabela de conversão considera duas colunas:
  • A primeira coluna contém a etiqueta (TAG) do campo que irá receber as informações na base de dados local
  • A segunda coluna contem o campo a ser aplicado ao registro MARC para transferir as informações à base de dados local
A conversão é realizada no momento da transferência de um registro para a base de dados local

Os formatos de conversão são mantidos na pasta /bases/cnv/. A lista de formatos existentes é chamada z3950.cnv


Tabela de caracteres de conversão MARC-8 para ANSI

Os servidores Z39.50 representam os caracteres especiais utilizando a codificação MARC-8 ou UTF-8, que considera duas posições para cada diacrítico. Esta tabela converte a codificação MARC-8 (dois caracteres para cada diacrítico) para ANSI (um caracter para cada diacrítico).

Se, depois de fazer uma pesquisa em um servidor Z39.50, houver combinações de caracteres que não pertencem ao idioma correspndente, deve-se incluir estas combinações na tabela de conversão de modo que os caracteres correspondentes sejam representados corretamente. 

A tabela de conversão de diacríticos é mantida na pasta bases com o nome de marc8-to-ansi.tab
