Captura de termos normalizados a partir de uma tabela

Esta opção do ABCD permite criar tabelas de valores que serão exibidos na entrada de dados correspondente. A forma como  a tabela será exibida dependerá do Tipo de entrada atribuída ao campo na FDT (Checkbox, Seleção simples, Seleção múltipla, Rádio, etc.). As tabelas de valores são armazenadas na pasta def da base de dados selecionada.

Para criar uma nova tabela informe na coluna Nome o nome da mesma, com a extensão .tab e clique sobre Browse.
Para visualizar uma tabela existente clique na imagem ou no nome da tabela correspondente.

Para editar uma tabela existente clique no nome da mesma e clique em Browse.

Para atribuir um campo a uma tabela existente, clique no nome da mesma e clique em atualizar FDT.

Para eliminar uma tabela existente clique no nome da mesma e clique em excluir. A tabela será deletada da pasta, mas permanecerá associada ao respectivo campo na FDT. Lembre-se, então, de excluir a entrada na FDT.

A lista de tabelas disponíveis exibe todos os arquivos com a extensão .tab localizados na pasta def da base de dados selecionada.

Criação/Edição de tabelas

Uma vez que o nome do arquivo tenha sido digitado na coluna Nome, clique em Browse. A seleção será exibida numa tabela com duas colunas: Código |  Termo

Se somente a coluna termo é preenchida, no procedimento de entrada serão exibidos os valores colocados nesta coluna, os quais serão transferidos ao campo correspondente, utilizando o método de seleção apropriado ao tipo de entrada do campo definido na FDT.

Se deseja transferir tanto o código como o termo associado, então a estrutura do campo que vai receber o valor da tabela deve contemplar 2 subcampos. No primeiro subcampo, será incluído o valor correspondente ao código e no segundo subcampo será colocado o valor correspondente ao termo.No procedimento de entrada serão exibidos somente os valores presentes na coluna termo, mas armazenará o código mais o termo, utilizando os subcampos correspondentes.

Ao terminar a edição da tabela, clique no botão Atualizar para salvar a tabela criada/modificada e atribuindo-a ao campo correspondente na FDT.

Dica: Para utilizar numa base de dados A uma tabela já definida para a base de dados B, o nome da Picklist deve ser precedido com o caminho correto onde está localizada a tabela que deseja usar.
