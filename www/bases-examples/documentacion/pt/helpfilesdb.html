Arquivos de ajuda para os campos da base de dados

Para definir as ajudas de cada campo na entrada de dados siga os seguintes passos:

  1. Selecione no menu Tabela de definição de campos (FDT) o campo para o qual pretende adicionar a ajuda 
  2. Realizada a seleção aparecerá no quadro as características do campo, conforme foi definido na tabela de definição de campos (FDT)
  3. Deverá clicar sobre Visualizar para visualizar a ajuda, se esta não estiver presente
  4. Clicar sobre Editar arquivo de ajuda para criar a ajuda ou modificar a já existente. Será apresentado o editor de HTML para que possa escrever o texto de ajuda
  5. Para terminar a edição da ajuda, clicar sobre Salvar ajuda para atualizar o arquivo
  6. Os arquivos de ajuda dos campos são guardados no diretório /bases/dbn/ayudas/tag_xxx.html, onde dbn é o nome da base de dados e xxx representa a etiqueta do campo a que corresponde a ajuda. Este nome é fornecido pelo sistema de forma automática e não deve ser modificado já que serve de referência para o procedimento de entrada da apresentação da ajuda.
  7. Una vez concluídos os arquivos de ajuda, deve editar a FDT da base de dados para marcar, na coluna correspondente, quais os campos para os quais foram estabelecidos arquivos de ajuda.
