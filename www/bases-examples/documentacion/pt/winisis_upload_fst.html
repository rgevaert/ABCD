Carregar FST

Se a etapa anterior foi completada, o ABCD já tem registradas a FDT e a FST da base de dados em processo de criação.

Use então a caixa Carregar .pft para localizar o formato de saída a ser atribuído à base de dados. Neste processo só é permitido carregar um formato, mas depois, através do módulo de Atualizar definições de base de dados poderá adicionar formatos.

Lembre-se que o formato selecionado tem que estar codificado de tal forma que a saída reproduza uma página HTML. Formatos que não contemplem esta condição irão gerar um erro de sintaxe no ABCD.
