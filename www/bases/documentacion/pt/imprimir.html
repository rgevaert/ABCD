Geração de relatórios

Este procedimento permite criar um formato de saída - ou utilizar um previamente criado - para enviar informações diretamente a um processador de texto.

Os registros podem ser selecionados de duas formas:

  1. Através de uma faixa de MFN's (de - até)
  2. Através de uma consulta à base de dados. Para isso clique no ícone de pesquisa e construa a expressão de pesquisa correspondente

Uma vez selecionados os registros, deve selecionar um formato existente ou criar um novo formato.

Para selecionar um formato , revise a lista que é exibida em Usar um formato existente e clique no formato que deseja utilizar. Em seguida, clique no ícone que exibirá uma janela com o formato selecionado. Quem conhece as regras para elaboração de formatos do CDS/ISIS, pode editar o formato para realizar mudanças ou ajustes; de outra forma, não modifique a saída, pois se gerar erros não obterá nenhum resultado ao gerar a impressão da listagem.

Clique em enviar para mandar o formato para a tela de criação da saída. Pode testar o formato selecionando os registros tanto por faixa de MFN's ou realizando uma pesquisa na base de dados. Uma vez realizada a seleção, clique em Mandar para processador de texto (para enviar a listagem a um processador de textos)  ou em Visualizar para ver o resultado do formato através de sua exibição na tela.

Para criar um novo formato, selecione da lista de campos aqueles que deseja incluir na saída. Para selecionar os campos, dê um duplo clique  sobre o nome ou arraste-os com o mouse e clique nas setas correspondentes. Pode utilizar as teclas [Ctrl] e [Shift] para selecionar os campos. As setas simples transferem os campos selecionados de uma janela a outra e as setas duplas transferem todos os campos.

Pode utilizar os botões Subir ou Descer para mudar a posição dos campos selecionados.

Ao terminar a seleção, clique em Criar formato e aparecerá na janela o formato criado. Se conhece a linguagem de formatação do CDS/ISIS, pode editar o formato para realizar mudanças ou ajustes; de outra forma, não modifique a saída, pois, se gerar erros, não obterá nenhum resultado quando ordenar a impressão da listagem.

Pode testar o formato selecionando os registros por faixa de MFN's ou através de uma na base de dados. Uma vez realizada a seleção, clique em Enviar para processador de texto (para enviar a amostra obtida a um processador de textos)  ou sobre Visualizar para ver o resultado do formato através de sua exibição na tela.

Para salvar o formato criado, dê um nome (utilizando letras e números unicamente e sem espaços em branco) e elabore a descrição com a qual se apresentará este formato na lista de formatos existentes.
