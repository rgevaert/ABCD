Funções de Manutenção

Permitem:

  1. Inicializar uma base de dados
  2. Eliminar uma base de dados
  3. Bloquear uma base de dados
  4. Desbloquear uma base de dados

Inicializar uma base de dados

Esta opção elimina - de forma permanente e irrecuperável - todos osregistros de uma base de dados.

Apagar uma base de dados

Esta opção elimina - de forma permanente e irrecuperável - todos os arquivos de dados, formatos e arquivos de definição de uma base de dados

Bloquear uma base de dados

Com esta opção o administrador pode bloquear o acesso a uma base de dados para efeitos de reorganização, quando seus índices devem ser regerados ou por qualquer outra situação que não permita a entrada de dados

Desbloquear uma base de dados

Com esta opção o administrador permite o acesso a uma base de dados que estava bloqueada para efeitos de manutenção ou por outro motivo.

Atribuir número de controle

Por meio desta opção, ABCD pode atribuir de forma automática um número de controle que se insere automaticamente desde um valor inicial.  Necessita que na FDT da base de dados esteja definido um campo do tipo Auto-incrementar
