Selecionar Base de Dados

Quando uma base de dados é selecionada, é informado o número total de registros que ela tem. Também haverá uma advertência se base de dados estiver bloqueada.

Se a base de dados estiver bloqueada, não pode ser acessada por nenhum operador até que o Administrador de Base de Dados faça o desbloqueio.

Para bloquear uma base de dados use a opção Utilitários->Bloquear Base de Dados do menu principal do ABCD.

Somente o Administrador do Sistema ou o Administrador da Base de dados podem bloquear ou desbloquear bases de dados.

Outra causa de bloqueio de base de dados é produzida quando é gerada uma condição de erro ao manipulá-la, por exemplo, um processo de geração de arquivo invertido não finalizado, ou erro na FST quando se tenta atualizar um registro. Esta situação também ocorre no momento de abrir a base de dados. Neste caso, como o bloqueio foi forçado por um Administrador,  deve-se primeiro desbloquear a base de dados e desbloquear os registros através  do menu de utilitários associado ao botão Utilitários da barra de ferramentas do procedimento de catalogação.

O bloqueio de registros é realizado comumente dentro do procedimento de entrada de dados como parte do mecanismo de proteção da atualização dos dados. Isto é: quando um usuário edita um registro, este é bloqueado para que não possa ser modificado simultaneamente por outro usuário. Quando o registro é atualizado, é desbloqueado, e volta a estar disponível para os outros usuários.

Se o ciclo de atualização não é finalizado, isto é, o registro não é salvo ou não se cancela a atualização (botões salvar ou cancelar da barra de ferramentas), o registro permanece bloqueado por um tempo (4 horas) para os outros usuários.

Existem duas formas para desbloqueio antes do prazo mencionado:

  1. Se o mesmo operador que bloqueou o registro, solicita de novo sua edição, então, ele pode atualizar o registro, ou seja, pode atualizar ou cancelar a edição para liberar o registro
  2. Outra forma de desbloquear os registros é utilizando a opção Desbloquear registros do menu Utilitários da barra de ferramentas. Somente o Administrador do Sistema ou Administrador da Base de Dados tem acesso a esta opção.
