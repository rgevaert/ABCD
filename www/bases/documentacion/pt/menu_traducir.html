Tradução de mensagens e páginas de ajuda

O ABCD utiliza no processamento multilingue dois tipos de componentes:

Mensagens e etiquetas

Contém uma tabela das etiquetas utilizadas para dialogar com o usuário, assim como as mensagens de alerta, de erro, e outras utilizadas na comunicação com o usuário.

Estas tabelas estão armazenadas nos diretórios bases/lang/xx onde xx corresponde ao código do idioma. Adicionalmente criou-se o código do idioma 00 que é a tabela de mensagens utilizada no desenvolvimento dos processos. Quando se seleciona um idioma de diálogo, primeiro é lida a tabela do idioma selecionado e depois a tabela correspondente 00 cuja finalidade  é completar as mensagens que não estão descritas no idioma selecionado. Desta forma os módulos podem gerar novas mensagens, sem  modificar as tabelas de mensagens atualizadas pelos usuários, nem mostrar mensagens em falta durante a execução dos processos.

Quando se pede para atualizar uma tabela de mensagens de um idioma em particular, a mesma se complementa com a tabela do idioma 00 para que o usuário possa traduzir as novas mensagens incorporadas ao ABCD.

Páginas de ajuda

Esta opção permite traduzir as páginas de ajuda que existem nos diferentes módulos do ABCD. Para facilitar a tradução é apresentada a lista das páginas disponíveis em cada módulo e o usuário pode selecionar:

  • Visualizar a página correspondente
  • Editar a página de ajuda
